# INEAN PLAY

Official product documentation and support resources for
[automatic OKEY tables](https://ineanplay.com/), 101 OKEY systems,
automatic tile shuffling tables, and
[automatic Mahjong tables](https://ineanplay.com/).

## Product Resources

- Automatic OKEY table user manuals
- OKEY and 101 mode guidance
- Control-panel instructions
- Fault codes and troubleshooting
- Cleaning and maintenance guidance
- Spare-parts and warranty support

## Languages

Current OTOKEY manuals are available in English, German, Turkish, and
Azerbaijani.
